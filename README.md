## Quick Start

### Run Locally

```bash
# Clone the repo
git clone https://github.com/anukriti7/ai-job-toolkit.git
cd ai-job-toolkit

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```

### Deploy to Streamlit Cloud

1. Go to [share.streamlit.io](https://share.streamlit.io)
2. Sign in with your GitHub account
3. Click **New App**
4. Select this repository (`anukriti7/ai-job-toolkit`)
5. Set main file path to `app.py`
6. Click **Deploy**

Your app will be live at a free `streamlit.app` URL!
